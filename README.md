# Development

- You'll need [`node`](https://nodejs.org/en) installed.
- Make a new day:
```
./new.sh 03
```
- Paste the example input in `/03/test`
- Paste the puzzle input in `/03/input`
- Write solution in `/03/index.mjs`
- Test solution with:
```
./test.sh 03
```
- Run on puzzle input with:
```
./run.sh 03
```
